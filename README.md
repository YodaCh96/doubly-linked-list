# Doubly linked list

The doubly-linked list differs from its single-linked sister primarily by the additional references to the preceding nodes of each list entry.In the context of this task, you are to apply your knowledge of static and dynamic data structures in general, as well as the technical and content-related differences between various types of lists in particular, in order to determine the appropriate data structure for different situations.
